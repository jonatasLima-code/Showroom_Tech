Showroom VR – Projeto WEB 3.0 METAVERSO
Unity Version: 2022.3 LTS

Descrição:
Projeto de um showroom virtual com 5 produtos exibidos em pedestais interativos.

Funcionalidades:

*Ambiente 3D com iluminação configurada

*Skybox aplicado

*5 objetos 3D principais

*Painéis informativos ao olhar para os produtos

*Configurado para Android (Meta Quest via OpenXR)

Execução no PC:

1- Abrir a cena principal

2- Pressionar Play

3- Utilizar mouse para olhar os produtos

4- Ao focar no produto, o painel informativo aparece

Configuração para Meta Quest:

Plataforma Android selecionada no Build Settings

*OpenXR ativado

*Meta XR Feature Group habilitado

*Scripting Backend: IL2CPP

*Target Architecture: ARM64
